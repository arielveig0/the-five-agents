# outputs/

**Path:** `outputs/` (repo root)
**Owner:** [[yuval-agent]] (write-only producer)
**Type:** Image directory (generated artifacts)

## Purpose
Destination for every image generated by Yuval via [[skill-nano-banana-2]]. The MCP server (`mcp-image`) writes here directly when `IMAGE_OUTPUT_DIR` env var points at this folder; otherwise Yuval moves the file from the MCP's default location.

## Naming convention
`<slug>-YYYYMMDD.<ext>` — e.g. `park-bench-reader-20260427.png`. Collision suffix `-2`, `-3` etc.

## What does NOT go here
- Hand-written / non-model files.
- Reference / inspiration images (those go to [[reference-folder]]).

## Current state
Empty (`.gitkeep` placeholder only).
